FAQ

What is this?

Codidact is a community initiative for developing a community-driven knowledge sharing website. We have some awesome ideas for that, but some communities that are interested in using our software are in need of an immediate solution. Hence we used an existing Q&A software. See our project homepage for more information.

Have an idea for a community?

Do you have a community that is in need of Q&A software? If you can bring a core group of users, we'll host it for you. Please suggest it on our Meta site by posting in the Site Suggestions category.

Contribute?

We're developing solutions in both Ruby on Rails (that's what's currently running these sites), and in C#. If you want to contribute, you can visit us on GitHub. See our project homepage for more information.
